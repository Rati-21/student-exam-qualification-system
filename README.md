# student-exam-qualification-system

A student exam qualification application developed as part of a Cloud Native Programming project.

The application calculates a student's weighted average using assessment marks and determines whether they qualify to write the examination.

## Technologies

- Java
- HTML5
- CSS3
- JavaScript
- Docker

## Features

- Calculates weighted averages
- Determines exam eligibility
- Input validation in the Java application
- Web-based interface
- Console-based Java application
- Docker containerization

## How The Calculations Work

- Test 1 - 30%
- Test 2 - 50%
- Assignment - 10%
- Project - 10%
